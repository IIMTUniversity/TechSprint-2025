## ü§ñ About TechSprint
TechSprint 2025 is a month-long innovation sprint focused on building **Artificial Intelligence** solutions that solve real-world problems. We encourage participants to leverage the power of Google Cloud, Gemini, and Flutter to create impactful MVPs.

## üìÖ Important Timeline
| Event Stage | Dates |
| :--- | :--- |
| **Project Creation Window** | After December 1, 2025 |
| **Sprint Starts** | December 5, 2025 |
| **Submission Deadline** | January 20, 2026 |
| **Evaluation Period** | January 21 ‚Äì January 30, 2026 |

## üéì Eligibility Criteria
* Projects must be **newly created** (Repository history must start after Dec 1, 2025).
* Open to all students and tech enthusiasts.
* Individual or team entries are welcome.

---

## üì¶ Mandatory Deliverables
To be eligible for judging, your submission **must** include:

* **Project Deck:** A PDF/PPT/Slides link explaining the problem and solution.
* **MVP & Code:** A live link to the MVP and the GitHub Repository.
* **3-Minute Demo Video:** A video (YouTube/Drive link) showcasing the UI and functionality.
* **Tech Stack List:** Documentation of Google Technologies & AI Tools used.
* **Solution Description:** A concise 100-word summary of your project.

## üõ†Ô∏è Google Technologies & AI Tools
Build your project using these industry-leading tools:

### **Google Cloud & Infrastructure**
* **Firebase:** Authentication, Firestore, Hosting.
* **GCP:** Cloud Run, App Engine, Storage.
* **Flutter:** For cross-platform UI.

### **AI & Machine Learning**
* **Gemini API:** State-of-the-art multimodal LLM.
* **Vertex AI:** Enterprise-grade ML platform.
* **TensorFlow:** Deep learning framework.
* **Dialogflow:** Conversational AI and Chatbots.
* **ML Kit:** On-device machine learning for mobile.
